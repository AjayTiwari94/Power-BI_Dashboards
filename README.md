# Data Jobs Dashboard (Power BI)

## Overview
This project focuses on analyzing global job market data for data-related roles using Power BI. The dataset contains job postings from multiple countries, cities, and companies, helping to understand hiring trends, salary distribution, and demand across the world.

---

## Dataset
The data represents job postings collected from online platforms (2025 dataset).

### Files:
- `job_postings_flat.csv` – Raw job-level dataset  
- `job_postings_monthly.xlsx` – Another file version of data  

### Key Information:
- Job roles (Data Analyst, Data Scientist, Data Engineer, etc.)
- Company and location (country & city)
- Salary details
- Job availability across regions
- Time-based trends (monthly data)

---

## Objective
- Analyze global demand for data-related roles  
- Identify top countries and cities with highest job openings  
- Compare salary trends across roles  
- Understand market distribution and hiring patterns
